# Funnel

The reader journey through the blog. Documents the stages, the surfaces that serve each stage, and the inferred / measured drop points.

```
Discovery (referrer, search, LinkedIn, direct)
  → First-read surface (post detail, landing)
  → Second-look (tag page, archive, series, related-posts component)
  → Subscribe / return (RSS subscribe, LinkedIn follow, repeat visit)
```

## Files

- `stages.md` — the five-stage map with surfaces, expected drop points, and known signals.
- `evidence.md` — anything we have concretely: Cloudflare analytics extracts, RSS subscriber counts, LinkedIn referral stats, top-N posts.
- `gaps.md` — points where the funnel narrative is inference rather than measurement; flagged so Stage 2 prescription doesn't act on unverified claims.

Per the no-unattributed-claims rule (workspace CLAUDE.md), bare conversion percentages without a source go in `gaps.md`, not `evidence.md`.
